## 🇬🇧 English

### About The Project
EcoRoute is an interactive mobile application designed to encourage sustainable daily habits. Built with a modern cross-platform tech stack, it provides users with the tools they need to track their eco-friendly actions, verify them using camera integrations, and seamlessly manage their progress.

### Core Technologies
* **Framework:** [React Native](https://reactnative.dev/) & [Expo](https://expo.dev/)
* **Backend & Database:** [Supabase](https://supabase.com/)
* **Navigation:** [Expo Router](https://docs.expo.dev/router/introduction/) & React Navigation
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Key Modules:** `expo-camera`, `expo-haptics`, `@react-native-async-storage/async-storage`

### Getting Started

#### Prerequisites
Make sure you have Node.js installed. You will also need the Expo CLI.

#### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/eliffozkan/ecoroute.git](https://github.com/eliffozkan/ecoroute.git)

```

2. Navigate to the project directory:
```bash
cd ecoroute

```


3. Install dependencies:
```bash
npm install

```


4. Start the development server:
```bash
npm run dev

```

### Proje Hakkında

EcoRoute, günlük yaşantımızdaki sürdürülebilir alışkanlıkları teşvik etmek ve takip etmek amacıyla geliştirilmiş interaktif bir mobil uygulamadır.
