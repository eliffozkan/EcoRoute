# EcoRoute

[![Open in Bolt](https://bolt.new/static/open-in-bolt.svg)](https://bolt.new/~/sb1-jglwkglr)

**EcoRoute** is an award-winning mobile application designed to gamify and encourage sustainable behaviors on university campuses. By turning daily eco-friendly actions into a rewarding experience, EcoRoute aims to build a greener future, one habit at a time.

### **✨ Key Features**

### **🚀 Tech Stack**

* **Frontend:** React Native, Expo Router  
* **Backend/Database:** Supabase  
* **Styling & UI:** Lucide React Native, React Native Reanimated  
* **Language Support:** Seamless English & Turkish i18n integration

### **🛠️ Getting Started**

1. Clone the repository and install dependencies:  
   npm install

2. Start the Expo development server:  
   npm run dev

3. Open with the **Expo Go** app on your phone or use a simulator.

**EcoRoute**, üniversite kampüslerinde sürdürülebilir yaşam alışkanlıklarını oyunlaştırarak teşvik eden ödüllü bir mobil platformdur. Günlük çevre dostu eylemleri eğlenceli ve ödüllendirici bir deneyime dönüştürerek daha yeşil bir gelecek inşa etmeyi hedefler.
